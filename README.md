Brainfuck Interpreter
=====================

Simple interpreter for the programming language [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck).

# Examples

Say "Hello [Name]"
```
++++++++[>++++[>++>+++>+<<<-]>+>+[<]<-]>>.>---.+++++++..+++.>.>+[,.]<+.
```

Reverse a String
```
+[->,]+[+.<]
```

Convert a word to lowercase.
```
++++[>++++++++<-]>[>,]<[[+<]>--[>]<]>>[.>]
```

Convert a word to uppercase.
```
++++[>++++++++<-]>[>,]<[[-<]>[>]<]>>[.>]
```
